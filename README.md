# 接口测试实战项目

## 项目简介
本项目是一个完整的接口测试与性能测试实战练习，包含：
- Flask 本地接口服务
- Pytest 接口自动化测试
- Postman 接口测试集合
- JMeter 性能压测脚本

## 技术栈
| 类型 | 工具/框架 |
|------|-----------|
| 接口服务 | Flask (Python) |
| 接口测试 | Pytest, Postman |
| 性能测试 | JMeter |
| 版本控制 | Git + GitHub |
